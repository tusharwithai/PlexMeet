# PlexMeet 
**AI-Powered Accessible Video Conferencing Platform**

## 🚀 About
PlexMeet is a real-time, bi-directional sign language translation tool designed to bridge the communication gap between hearing-impaired individuals and the general population. It integrates Indian Sign Language (ISL) recognition directly into a video meeting interface (similar to Zoom/GMeet).

## 🛠 Tech Stack
- **Core AI:** Python, MediaPipe (Holistic), LSTM (Deep Learning)
- **Computer Vision:** OpenCV, NumPy
- **Interface:** React.js / Streamlit (In Progress)
- **Real-Time Communication:** WebRTC
